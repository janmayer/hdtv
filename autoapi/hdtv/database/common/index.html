Return a list of nuclides with given properties (Z, A, symbol, name)

e.g.: Nuclides(A=197, symbol=”Au”) returns [Au-197]

Nuclides(symbol=”Au”) or Nuclides(name=”gold”) or Nuclides(Z=79) return list of all gold nuclides
