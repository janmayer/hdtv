FitPosAssign(args)

Assign a nominal value for the positon of peaks

Peaks are specified by their id and the peak number within the fit. Syntax: id.number If no number is given, the first peak in the fit is used.

FitPosErase(args)

Erase nominal values for the position of peaks

Peaks are specified by their id and the peak number within the fit. Syntax: id.number If no number is given, the all peaks in that fit are used.

FitPosMap(args)

Read a list of energies from file and map to the fitted peaks.

The spectrum must be roughly calibrated for this to work.
